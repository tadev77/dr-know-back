# Dr Know - Back-end

## Instalação

Use a versão 21 do node e instale com seu gerenciador de pacotes preferido
```
npm i
```

Crie um arquivo `.env` tal como está em `.env-example`

Faça as migrações e seeds de db

```
npm run migrate:latest
npm run seed:run
```

Rode a aplicação

```
npm start
```
